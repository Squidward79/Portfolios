## 2. Rhythm Goblin
- タイトル：Rhythm Goblin
- ジャンル：２Dリズムゲーム
- プラットフォーム：PC
- 製作期間：２０１７．１０ ～　２０１７．１２
- 使用ツール：Unity3D/C#
- 製作人数：1人

＊説明
リズムに合わせて攻撃をすると、敵を吹き飛ばします。
攻撃の時、ゴブリンの前にある円形と敵の体にあるオレンジ色の円形の距離を計算し、判定します。判定がいいほど、スコアが高く増えます。
赤色のスキルゲージを集まってステージが始まる前に選択したスキルを使えます。

学校の課題で製作したゲームプロジェクトです。
グラフィックのリソースやサウンドは学校で提供されました。
